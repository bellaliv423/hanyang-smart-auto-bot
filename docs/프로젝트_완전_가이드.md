## Phase별 완료 현황

### Phase 1: HY-ON LMS 로그인 자동화 (완료)
- **파일**: `scrapers/hyon_login.py`
- **핵심**: 한양 시스템은 **2단계 로그인** 필요!
  1. 포털 SSO: `portal.hanyang.ac.kr` → `input#userId` + `input#password`
  2. LMS 별도: `api.hanyang.ac.kr` → `input#uid` + `input#upw` + `button#login_btn`
- **비밀번호 변경 팝업**: `#btn_cancel` ("다음에 변경") 자동 처리
- 실행: `python scrapers/hyon_login.py`

### Phase 2: LMS 스크래핑 (완료)
- **파일**: `scrapers/course_scraper.py`
- **과목별 메뉴 URL 패턴** (모든 과목 동일):
  - 수업계획서: `/courses/{id}/assignments/syllabus`
  - 게시판: `/courses/{id}/external_tools/132`
  - 주차학습: `/courses/{id}/external_tools/140`
  - 강의자료실: `/courses/{id}/external_tools/3`
  - 출결현황: `/courses/{id}/external_tools/138`
- 실행: `python scrapers/course_scraper.py`

### Phase 3: AI 학습 보조 (완료)
- **파일**: `study_assistant/assistant.py`
- **Claude API**: `claude-sonnet-4-5-20250929` 모델 (timeout=120s 필수!)
- **기능**:
  - `--mode syllabus` — 수업계획서 스크린샷 분석 (3개 언어)
  - `--mode weekly` — 주차학습 분석
  - `--mode preview` — 예습 자료 생성
  - `--mode review` — 복습 자료 생성
  - `--mode explain --query "주제"` — 특정 주제 해설
- 실행 예시:
  ```bash
  python study_assistant/assistant.py --mode syllabus
  python study_assistant/assistant.py --course 196594 --mode explain --query "회귀분석"
  ```

### Phase 4: WhatsApp 리마인더 (완료)
- **파일**: `reminders/whatsapp_reminder.py`
- **알림 3종**:

  2. 수업 30분 전 — 개별 수업 알림 + 예습 포인트
  3. 일요일 20:00 — 주간 시간표

